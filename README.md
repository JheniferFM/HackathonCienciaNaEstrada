# Nexus Mobility AI Project

## 📌 Visão Geral

O Nexus Mobility AI Project é uma plataforma inteligente de mobilidade urbana desenvolvida para auxiliar cidadãos, gestores públicos e operadores de transporte na análise de deslocamentos, lotação, segurança e planejamento de rotas.

A solução integra recursos de visualização geográfica, inteligência artificial e análise de dados para oferecer uma experiência moderna e interativa.

## 🎯 Funcionalidades Principais

### 🗺️ Mapa Inteligente

Visualização de rotas e conexões entre regiões administrativas.

### 🤖 Assistente IA

Auxilia usuários com informações sobre mobilidade urbana e transporte público.

### 🚍 Previsão de Lotação

Estimativas de ocupação em linhas de transporte público.

### 🛡️ Segurança Urbana

Monitoramento e exibição de informações relacionadas à segurança no deslocamento.

### 📊 Dashboard Governamental

Painel estratégico para análise de indicadores de mobilidade.

### 🚗 Sistema de Caronas

Integração de soluções de mobilidade compartilhada.

## ⚙️ Como Executar Localmente

### Pré-requisitos

* Node.js (versão LTS recomendada)
* npm

---

### 1. Clonar o Repositório

```bash
git clone <url-do-repositorio>
cd Nexus-Mobility-AI-Project
```

### 2. Instalar Dependências

```bash
npm install
```

### 3. Executar Ambiente de Desenvolvimento

```bash
npm run dev
```

Após iniciar o servidor, o Vite exibirá uma URL semelhante a:

```text
http://localhost:5173
```

Abra a URL no navegador.

---

## 📦 Gerar Build de Produção

```bash
npm run build
```

Os arquivos otimizados serão gerados na pasta:

```text
dist/
```
